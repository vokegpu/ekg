# Code of Conduct 🐄

### Social 🐈

EKG is an open-source project hosted by VokeGpu, EKG is `she/her`.

Maintaing EKG is a voluntary work, BUT VokeGpu does not allow any of these things:
- AI-generated codes ARE NOT allowed.
- Racism, transphobism, free hate any group of peoples, religious attack, and political-ideology attack ARE NOT allowed.
- Off-topic discussions ARE NOT allowed.
- Making PR(s) MUST follow the [coding-guide style](https://github.com/vokegpu/code-of-conduct-and-style-guide).
