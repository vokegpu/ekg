# 🐄 EKG 🐈

EKG is a descriptor-based low-latency modular UI-toolkit for desktop-apps, mobile-apps, and high-performance apps. 

Platforms: SDL2, GLFW.  
GPU APIs: OpenGL3+, OpenGLES3.

**Note: EKG is not stable yet.**
